# BRAIN TRUST _By_ Magnum PI

### Installation
`git clone https://github.com/brice1382/braintrust.git`

### Install Dependencies
`npm i`

### Run
`npm start`

### Client Code
Some of the libraries and frameworks that I used:

* AngularJS
* Bootstrap 3.3.7
* Sweetalert
* Animate.css
* Font Awesome.
* SCSS Stylesheets

### Server Code
* ASP.NET MVC 5
* Entity Framework
* WebApi 2

_Server_

* Windows Server 2012 R2

_Database_

* MS SQL Server

_Web Server_

* IIS 8
